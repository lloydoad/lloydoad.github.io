PROJECTS

SPACE INVADERS | JAVA

Space Invaders was just a recreation of the classic space invaders game. Using arrays and arraylists, the (x,y) position of each sprite is constantly monitored at each refresh. This also allowed collisions to be possible by calculating when different (x,y) values intersected. These arrays were also recalculated at each frame to enable the deletion of sprite objects which were no longer in the screen. Using File I/O, data like paused state and highscores where written and read from when necessary.

SPADES | JAVA

Spades was my first venture into making games with Java. The idea of object-oriented programming was the best framework for making a game that required multiple objects and protected classes. Concepts like polymorphism and inheritance enabled classes like Human players and A.I. to get basic functionality from a single player class. Other features like abstract classes and interfaces allowed game action functions to be shared among several unrelated classes.
