PROJECTS

TELL | SWIFT

Tell is an iOS application that automatically translates messages between recipients. Users can also set languages preferences. To simulate a smooth paging effect between the main views, all aspects of the ViewControllers were programed. Yandex API was accessed through REST API to enable text message translations. Other technologies used were Firebase and Realm Database to ensure the data was constantly uploaded and refreshed without interfering with the main thread.

YOUTUBE CABLE | PYTHON

Youtube cable is meant to be a python web application that uses several youtube content to create an online cable network. The application would run through youtube live API and use resources like Heroku to ensure an always-online state. This project is currently on hold due to other responsibilities and unfinished projects.
